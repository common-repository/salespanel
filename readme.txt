Add Salespanel tracking code and configuration options on you Wordpress website without writing any code.


== Description ==

[Salespanel](https://salespanel.io/)'s plugin for Wordpress allows you to set up Salespanel's website SDK in no time. Start capturing and tracking your leads out of the box without modifying your theme.

Salespanel's plugin for Wordpress offers following configurations:

* Enable automatic lead capture.
* Enable lead capture on web form submit.
* Disable lead web tracking (Salespanel tracks your visitors by default).

To get started, register on [Salespanel](https://salespanel.io/) and follow the [installation instructions](https://wordpress.org/plugins/salespanel/#installation).


== Installation ==

In-depth guide: [Install Salespanel's plugin for WordPress to start qualifying your leads under 10 minutes](https://support.salespanel.io/hc/en-us/articles/900003770426-Salespanel-s-plugin-for-WordPress-Start-qualifying-your-leads-under-10-minutes).

#### Install and activate the plugin ####

You can install Salespanel'a plugin for Wordpress in two ways:

= a. Installation from WordPress =

1. Visit **Plugins menu > Add New**.
2. Search for **Salespanel**.
3. Install and activate the Salespanel plugin.

= b. Manual installation =

1. Alternatively, you can download the zip file from [WordPress Plugin Directory](https://wordpress.org/plugins/salespanel/), extract it and upload the entire `salespanel` folder to the `/wp-content/plugins/` directory.
2. Visit **Plugins menu**.
3. Activate the Salespanel plugin.

#### After activation ####

1. Click on **Settings** link on the Salespanel plugin or in the **Settings** menu, find **Salespanel**.
2. Add your Salespanel Client ID in the field provided. To get the client ID required to activate tracking your visitors, please visit [here](https://salespanel.io/tracking-code/).
3. You can set up the settings for lead tracking options according to your requirement.

Please feel free to contact Salespanel Support team for any queries.
